Términos de Servicio y Política de Privacidad

1. Introducción

Esta aplicación es únicamente para pruebas y desarrollo. No está destinada a uso comercial ni al manejo de datos de usuarios reales. El acceso y uso de esta aplicación implica la aceptación de los siguientes Términos de Servicio y Política de Privacidad.

2. Uso permitido

La aplicación puede ser utilizada únicamente con fines de:

  • Pruebas técnicas de integración con APIs (como TikTok, Facebook, Instagram, etc.).
  • Desarrollo personal o en entornos de prueba.
  • No se permite su uso en entornos de producción ni comerciales.

3. Limitación de responsabilidad

El desarrollador de esta aplicación no se hace responsable de:

  • Fallas técnicas derivadas del uso de APIs externas.
  • Pérdida de datos, errores de ejecución o problemas relacionados con configuraciones del usuario.
  • Cualquier uso indebido fuera del contexto de pruebas.

4. Política de Privacidad

La aplicación no recopila, almacena ni procesa datos personales de usuarios reales. En el contexto de pruebas, puede acceder temporalmente a:

  • Información básica de perfil (dependiendo de la API en uso, por ejemplo: nombre de cuenta o identificador).
  • Permisos de publicación en cuentas propias, con el único fin de probar la funcionalidad de la API.

5. Uso de la información

Cualquier información accedida se utiliza exclusivamente para:

  • Pruebas de funcionalidad de APIs.
  • Validación del correcto funcionamiento de la aplicación.
  • No se almacena ni comparte información personal de manera permanente.

6. Compartir información

No compartimos, vendemos ni distribuimos información de usuarios a terceros.

7. Seguridad

Esta aplicación de prueba no almacena datos sensibles. El acceso a las APIs se limita a los permisos otorgados temporalmente para testing.

8. Cambios

Estos Términos y la Política de Privacidad pueden actualizarse ocasionalmente. Los cambios se reflejarán en esta página.

9. Contacto

Para cualquier consulta relacionada con esta aplicación de pruebas, puedes escribir a: tu-email@ejemplo.com
